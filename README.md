# home-town

## 这是三小

一个练手项目，在2021暑假实习期间作为项目设计的前端部分，主要基于[vue-manage-system](https://github.com/lin-xin/vue-manage-system)实现。

至于项目效果可以移步vue-manage-system查看，由于后端部分尚未发布，不建议单独下载运行。
